# Online Food Ordering System

## Description
This is a web-based food ordering system developed using PHP and MySQL. It allows users to browse food items, add them to cart, and place orders. Admin can manage menu and orders.

## Features
- User registration and login
- Add to cart functionality
- Order placement
- Admin dashboard for managing orders


- ## How to Run the Project

1. Install XAMPP
2. Copy project folder to htdocs
3. Start Apache and MySQL
4. Import database.sql into phpMyAdmin
5. Open browser and go to:
   http://localhost/online-food-ordering-system
