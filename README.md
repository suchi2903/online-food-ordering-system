# YummiYo - Online Food Ordering System

## Description
YummiYo is a web-based food ordering system developed using PHP and MySQL.  
It allows users to register, login, browse food items, add them to cart, and place orders.  
An admin panel is also included to manage menu items and customer orders.


## Features
- User Registration & Login
- Secure Authentication System
- Browse Food Menu
- Add to Cart Functionality
- Order Placement
- Admin Dashboard (Manage Orders & Menu)


## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (Apache Server)

## How to Run the Project

1. Install XAMPP
2. Start **Apache** and **MySQL**
3. Copy the project folder into: C:\xampp\htdocs\

4. Open phpMyAdmin:

http://localhost/phpmyadmin

5. Create a database:

yummiyo_db

6. Import the `.sql` file into the database
7. Open browser and run:

http://localhost/yummiyo/
